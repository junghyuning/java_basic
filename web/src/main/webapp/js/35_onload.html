DOM(Document Object Model) - onload Event Listener

DOM : document 객체를 기반으로 제공되는 HTML태그에 대한 자바스크립트 객체

요청 웝프로그램의 실행 결과를 웹문서로 전달받아 브라우저의 응답영역에 출력

브라우저는 DOM Parser 를 사용하여 웹문서의 html 태그를 DOM Tree로 생성 - 자바 스크립트는 DOM Tree의 HTML 태그를 Element 객체로 제공받아 사용 가능

자바스크립트를 사용하여 태그에서 이벤트가 발생될 경우 이벤트 처리 함수를 사용하여 태그를 element 객체로 제공받아 조작 가능 - 동적 페이지(DHTML:DynamicHTML) 구현

동적인 페이지는 반드시 DOM Tree가 완성된 후 동적인 페이지 구현이 가능함.
