Mouse Event

click : 태그에서 마우스 버튼을 누른 경우 발생하는 이벤트

dbclick(더블클릭) : 태그에서 마우스 버튼을 두번 연속으로 누른 경우 발생하는 이벤트

mousedown : 태그에서 마우스 버튼을 누르는 '순간에' 발생하는 이벤트

mouseup : 태그에서 마우스 버튼을 '놓는' '순간에' 발생하는 이벤트

mouseover : 태그에 마우스 커서가 진입하는 순간 발생하는 이벤트 - 자식태그 포함

mouseout : 태그에 마우스 커서가 벗어나는 순간 발생하는 이벤트 - 자식태그 포함

mouseenter : 태그에 마우스 커서가 진입하는 순간 발생하는 이벤트 - 버블링단계 제외

mouseleave : 태그에 마우스 커서가 벗어나는 순간 발생하는 이벤트 - 버블링단계 제외

hover : 태그에 마우스 커서가 진입 or 벗어나는 순간 발생하는 이벤트

mousewheel : 태그에서 마우스휠이 움직이는 동안 발생하는 이벤트
